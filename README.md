# Reflect_Cube
3d billiard.

## Description
フィールド
- 立方体

ボール
- 上から1,3,6,10の4層のピラミッド型で合計20個プラス白玉

ルール
- ボールの数字がボールのHPになっている．
- 壁に当てるとボールのHPが１ずつ減る．
- ボールのHPを0にするとボールが消える．
- ボールを消した人がボールの最大HP分を得点として獲得する．
- ボールを打つのは一回づつで交代

人数
- 2~4人
- 得点が高い人が勝ち
